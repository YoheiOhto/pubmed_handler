以下の手順で実験一覧を表示してください。

## ステップ1: 実験ディレクトリ一覧
```bash
lsx
```

## ステップ2: 各実験のステータスを確認
最新5件について `logs/{exp}/latest/run_metadata.yaml` を読み、ステータスを確認する。

## ステップ3: 結果サマリー付きで表示
以下の形式で出力する：

| ID | 実験名 | ステータス | 主要指標 | 日時 |
|----|--------|----------|---------|------|

ステータスの絵文字：
- COMPLETED → ✅
- FAILED / TIMEOUT → ❌
- RUNNING → 🔄
- 未実行（ログなし）→ ⏸️
